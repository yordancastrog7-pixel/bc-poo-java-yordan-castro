# Semana 04 - Herencia en Java
Este ejercicio modela productos de una cafetería usando herencia.

## Archivos
- ProductoCafe.java → Clase padre
- CafeCaliente.java → Subclase
- CafeFrio.java → Subclase
- Main.java → Demostración de polimorfismo
- JERARQUIA.md → Diagrama y documentación

## Conceptos aplicados
- Herencia con `extends`
- Uso de `protected`
- Sobrescritura con `@Override`
- Constructores con `super()`
- Polimorfismo en arreglos
