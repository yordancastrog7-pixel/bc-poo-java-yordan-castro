# Arrays vs Colecciones en Java

- Un **array** es una estructura de tamaño fijo: se define al crearse.
- Una **colección** puede crecer o reducir dinámicamente.
- Arrays permiten acceso por índice muy rápido.
- Colecciones ofrecen métodos útiles: `add`, `remove`, `contains`, `size`.
- Usa **arrays** cuando conoces la cantidad exacta de elementos.
- Usa **colecciones** cuando necesitas flexibilidad y operaciones avanzadas.
- Las colecciones principales: `List`, `Set`, `Map`.
