# Semana 08: Colecciones y Programación Genérica - Café Aroma

## Objetivo
Implementar sistemas usando colecciones (ArrayList, HashMap, Set) y genéricos en el dominio Cafetería "Café Aroma".

## Estructura
- `src/com/cafearoma/` -> código fuente (modelo, servicio, util)
- `2-practica/` -> ejercicios prácticos (carpetas)
- `1-teoria/` -> material teórico
- `PRUEBAS.md` -> casos de prueba

## Cómo compilar y ejecutar
Desde la raíz `semana-08/`:
```bash
# Compilar
javac -d bin src/com/cafearoma/**/*.java

# Ejecutar
java -cp bin com.cafearoma.Main
