# Semana 06 - Abstracción e Interfaces

Implementación de clase abstracta, interfaces y polimorfismo aplicado al dominio
"Cafetería Aroma".

Incluye:

- Clase abstracta `ProductoAbstract`
- Interfaces `Vendible` y `Descontable`
- Clases concretas: `Bebida`, `Postre`, `Sandwich`
- Programa demostración en `Main.java`
- Documento de análisis en `ANALISIS.md`
